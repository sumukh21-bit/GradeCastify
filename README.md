# GradeCastify

## Overview
GradeCastify is a full-stack web application designed to help students manage and understand their academic performance. Users can input course data, track progress, and receive predicted grades using a machine learning model.

---

## Features
- User authentication (sign up and login)
- Dashboard for viewing academic performance
- Add and manage course data
- Profile customization (study habits, stress, sleep, etc.)
- Note editor for personal use
- Machine learning grade prediction
- Backend API for data processing

---
